# ThemeForge v0.2.0

## Updated

- Added DOCX transcript import using the Python standard library.
- Added basic RTF transcript import using the Python standard library.
- Added a bounded clustering path for larger transcript files.
- Added regression tests for DOCX, RTF, and larger transcript analysis.
- Preserved the qualitative workflow from v0.1.0: suggested themes, keyword
  labels, and ranked quote evidence.

## Verification

- Unit tests pass with parser, analyzer, export, DOCX, RTF, and bounded
  clustering coverage.
- Public dataset QA was run against a CC BY 4.0 higher education interview
  transcript dataset from figshare.
- Windows portable zip was rebuilt and launch-tested.
