# ThemeForge v0.3.0

## Updated

- Added a contextual coding model for ASD and 4-H interview transcripts.
- Added 4-6 word theme labels for the ASD/4-H context, including barriers,
  sensory and transition challenges, training accommodations, useful
  activities, smaller-club inclusion, gradual involvement, and peer/staff
  support strategies.
- Changed the default quote setting to list all matching quote options for each
  theme. Use a positive quote limit when a shorter preview is needed.
- Added multi-label quote evidence, so one quote can appear under multiple
  themes when it supports multiple analytic codes.
- Added stable theme colors to Markdown, CSV, JSON, and the desktop detail view.
- Added regression tests for the ASD/4-H sample behavior and color-coded output.

## Verification

- Unit tests pass for transcript parsing, DOCX/RTF import, contextual coding,
  all-quote evidence, color-coded themes, export, and bounded clustering.
- The ASD/4-H sample was run through the CLI and now returns contextual themes
  instead of vague one-word labels.
- Windows portable zip was rebuilt and launch-tested.
