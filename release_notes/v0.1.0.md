# ThemeForge v0.1.0

## Purpose

This release rebuilds the project for qualitative interview and focus group
analysis instead of large-corpus phrase mining research. The app helps
researchers review transcript text, generate suggested thematic groupings, and
collect quote evidence under each suggested theme.

## Updated

- Added a Windows-friendly desktop app using Python and Tkinter.
- Added a command-line analyzer for plain text transcripts.
- Added transcript parsing that preserves speaker labels when lines use
  `Speaker: text`.
- Added quote-unit extraction, suggested theme clustering, keyword labels, and
  ranked quote evidence.
- Added Markdown, CSV, and JSON exports.
- Added a Windows portable build script that creates a zipped app folder.
- Replaced the original AutoPhrase README with project documentation for the
  qualitative coding workflow.

## Important Limitations

- Theme names and quote matches are machine-generated suggestions.
- Researchers must review the original transcript context before interpreting
  or reporting findings.
- v0.1.0 uses an embedded standard-library lexical model. A later release can
  add local embedding or small language model support after model licensing,
  package size, and offline performance are verified.
