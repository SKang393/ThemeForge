# ThemeForge v1.0.0

## Updated

- Promoted ThemeForge to a 1.0.0 release after sequential v0.5.0, v0.6.0,
  v0.7.0, v0.8.0, and v0.9.0 checkpoints.
- Reworked the desktop app into a minimal professional three-panel workspace:
  input controls, color-coded themes, and quote evidence.
- Added a cleaner header with version, import, analyze, and export actions.
- Added a compact input panel for selected transcripts, central theme, theme
  count, and quote count.
- Added concise theme rows showing theme label, quote count, and focus
  alignment.
- Added cleaner quote evidence formatting with source file, line, speaker,
  quote ID, relevance, and color-coded theme marker.
- Added UI view-model helpers for stable professional wording and styling.
- Kept the app dependency-light: Python standard library plus Tkinter, with no
  new runtime dependencies.

## Verification

- Unit tests cover analysis behavior, imports, CLI multi-file use, release
  metadata, and UI view-model helpers.
- Compile checks cover the ThemeForge package, tests, and GUI launcher.
- CLI smoke checks were run against English and Korean transcript samples.
- Windows portable zip was built and launch-tested before release publication.
