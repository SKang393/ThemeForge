# ThemeForge v0.4.0

## Updated

- Renamed the program from AutoPhrase Interview to ThemeForge.
- Added multi-file transcript analysis for related interviews or focus groups.
- Added an optional central-theme field for prioritizing a researcher focus
  such as accessibility, teacher perspective, parent burden, autism, or a brief
  study topic.
- Preserved source file names and transcript line numbers in quote evidence.
- Added validation metadata for each suggested theme: evidence count, displayed
  quote count, source count, speaker count, central-theme alignment, and review
  status.
- Added a validation framework section to Markdown reports.
- Added color-coded transcript evidence with theme IDs, colors, source files,
  speakers, quote IDs, and line numbers.
- Added initial Korean transcript tokenization for first-pass theme and quote
  suggestions.
- Tightened RTF import so font-table and color-table metadata do not become
  suggested themes.
- Tightened contextual ASD/4-H detection so autism-only transcript data does
  not receive 4-H-specific labels.
- Filtered common PDF/OCR extraction artifacts, including participant IDs and
  glued discourse tokens, from keyword scoring.
- Updated the desktop app for multi-file selection, central-theme entry,
  color-coded theme lists, source-aware quote details, and validation notes.
- Updated the CLI to accept multiple input files and `--central-theme`.
- Updated README credits and research basis, including links to the original
  AutoPhrase project and thematic-analysis validation sources.

## Verification

- Unit tests cover transcript parsing, DOCX/RTF import, bounded clustering,
  contextual ASD/4-H coding, source-aware multi-file analysis, central-theme
  prioritization, validation metadata, color-coded Markdown export, Korean
  tokenization, CLI multi-file use, RTF metadata stripping, and PDF/OCR artifact
  filtering.
- Public transcript QA was run locally against ignored public-data files from
  DaRUS, King's College London, and figshare / De Montfort University.
- Windows portable zip should be rebuilt from this version and launch-tested
  before release publication.
